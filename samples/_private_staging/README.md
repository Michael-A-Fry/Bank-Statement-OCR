# Private staging — drop real statements here (stays on this machine)

This folder stays **local**. Anything you put here remains on this machine and is
never shared or distributed. Only this README travels, so the folder exists and
explains itself.

Use it for **real / personal bank statements** you want tested against the engine
but must not publish — e.g. your own everyday statements.

## How to test a file dropped here
Open the app and upload it on the **Convert** tab. Outputs stay local too.

## Safety
- The private-staging folder is excluded from any distributed or shared copy (this
  README excepted).
- Nothing in here is read by the test suite or included in any package.
